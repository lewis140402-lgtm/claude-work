# Paper Trading Performance Log

**This is paper money only. No real funds are ever connected to this
project.**

No entries yet — this file is regenerated by `weekly_update.py` once you've:
1. Created a free Alpaca paper trading account and generated API keys
2. Filled in `.env` with those keys
3. Run `python3 fetch_data.py SPY` and `python3 backtest.py SPY` to get a
   real (not synthetic) 5-year backtest first — see `backtest_report.md`
   and read the verdict honestly before starting the live paper loop
4. Started running `run_daily.py` (daily) and `weekly_update.py` (weekly)

Once running, this file becomes a table: date, paper equity, % return since
start, and the SPY buy-and-hold benchmark's return over the same window —
so you can see with your own eyes whether the strategy is actually adding
anything, not just trust a claim that it is.
